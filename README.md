# Desenvolvido uma aplicação, com Front-End e Back-End

## Cloning the Project


```bash
  git clone https://github.com/markinascimento/Project-Front-Back
```

## Open folder backend

```bash 
cd backend
```
## Running project backend

```bash 
yarn dev
```
## Open folder frontend

```bash 
cd frontend
```
## Running project frontend

```bash 
yarn start
```
